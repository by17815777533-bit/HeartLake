## 4. 常见问题

| 现象 | 优先排查 |
|---|---|
| 前端连不上 API | `PUBLIC_ORIGIN`、`API_BASE_URL`、`VITE_API_BASE_URL` 是否指向当前公网入口 |
| WebSocket 一直断开 | `ws://121.41.195.165/ws/broadcast` 是否正确，token 是否有效 |
| 列表显示空白 | 响应里是否缺 `items / total / page / page_size` |
| 页面显示默认值 | 是否还有数据源或页面层在吞异常 |
| 推荐或 AI 没结果 | 接口是否显式返回失败或 `degraded` |
| 管理端图表不刷新 | 先看统计接口 200，再看 WebSocket 状态 |
