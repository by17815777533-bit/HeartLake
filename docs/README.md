## 3. 当前项目真实语义

这几条是当前有效的硬约束：

- 好友关系由亲密度自动判定，`accept/reject/pending` 相关入口不可用。
- 临时好友只允许由石头/纸船互动自动建立；手动直连和手动升级永久好友不可用。
- 账号停用相关路由必须显式提供 `confirmation`，不接受空请求体。
- 移动端启动页只有在会话验证成功后才会进入首页；失败不会再伪装成已登录。
- 情绪日历、热力图、情绪脉搏只认真实 `score/raw_score/mood` 数据，不再用默认中性值兜底。
- 管理端和移动端列表页要求显式 `total/page/page_size/has_more` 集合契约，不再猜分页。
- Edge AI 管理配置对非法值做硬拒绝，不再把坏输入静默改回默认值。
