# 管理端代码地图与运营链手册

本文按当前 `admin/src/` 代码结构描述管理端入口、页面、状态、实时链和运营工作流。

## 1. 入口与路由

### 关键入口

- 路由：`admin/src/router/index.ts`
- 布局：`admin/src/layouts/MainLayout.vue`
- 全局状态：`admin/src/stores/index.ts`
- HTTP 基座：`admin/src/api/index.ts`
- WebSocket：`admin/src/services/websocket.ts`

| `/moderation` | `Moderation.vue` | admin |
| `/reports` | `Reports.vue` | admin |

## 3. 主布局与顶栏链

`MainLayout.vue` 当前承担：

- 顶部导航和工作台品牌区
- 路由搜索
- 管理员头像菜单
- 顶栏实时状态摘要
- Dashboard 外层的 `router-view`

布局页当前主动维护两条头部数据链：

- `getAdminInfo()`
- `getRealtimeStats()`

失败时会保留 warning，而不是静默变空。

## 4. 当前状态管理

`stores/index.ts` 当前存这些全局状态：

- `token`
- `userInfo`
- `isDark`
- `loadingCount`
- `isGlobalLoading`

当前存储策略：

- `admin_token` 和 `admin_user_info` 放 `sessionStorage`
- 暗色模式放 `localStorage`
- token 时间戳 `admin_token_ts` 用于前端 24 小时启发式校验

## 5. HTTP 通信链

`api/index.ts` 当前做的事情：

- `VITE_API_BASE_URL || '/api'` 作为 base URL
- 请求前注入 Bearer token
- 生成请求指纹并自动取消重复请求
- 聚合全局 loading
- 业务层 `code != 0/200` 视为失败
- 401 和业务认证错误触发统一登出
- GET 遇到 `408/429/500/502/503/504` 自动指数退避重试

## 6. WebSocket 实时链

`services/websocket.ts` 当前关键行为：

- 从 Pinia store 读取 token
- `VITE_WS_URL` 或当前 host 组装 `ws://.../ws/broadcast`
- 心跳 `30s`
- 最大重连 `10` 次
- 未知消息类型转成 `socket_error`

当前白名单类型包括：

- `stats_update`
- `new_report`
- `new_moderation`
- `broadcast`
- `user_online`
- `user_offline`
- `pong`
- `auth_success`
- `new_stone`
- `ripple_update`
- `boat_update`
- `stone_deleted`
- `friend_removed`
- `new_notification`
- `socket_status`
- `socket_error`

## 7. 页面职责

### Dashboard

- 聚合平台实时数据
- 使用 `useDashboardData`、`useDashboardLoaders`、`useChartOptions`
- 图表失败时显示 stale / warning，而不是直接清空

### Users

- 用户列表
- 用户详情
- 封禁 / 解封 / 状态切换
- 高级推荐查看入口

### Content

- 石头、纸船、内容总览
- 当前内容水位和治理动作

### Moderation

- 待审核内容
- 审核通过 / 拒绝
- 审核历史

### Reports

- 举报列表
- 举报详情
- 求助事件处理

### SensitiveWords

- 风险词条维护

### Logs

- 管理动作和系统日志查看

### Settings

- 高权限配置管理
- 广播、系统参数、模型参数

### EdgeAI

- EdgeAI 状态
- 节点信息
- 相关推理和预算指标
