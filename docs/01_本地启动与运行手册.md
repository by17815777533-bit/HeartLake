# 本地启动与运行手册

当前仓库支持两种运行方式：本地联调和直连云端现网。

## 1. 环境要求

- Flutter SDK
- Node.js 18+
- Docker 和 Docker Compose
- CMake / Ninja / gcc 或 clang
- git / curl / jq

## 2. 推荐启动方式

推荐用 Docker 启动后端、网关、数据库、Redis 和管理端，再用 Flutter 单独跑移动端。

```bash
./scripts/docker-up.sh server-lite
cd frontend
flutter pub get
flutter run
```

## 3. 单模块启动

### 后端

```bash
cd backend
cmake -S . -B build-2c2g -DCMAKE_BUILD_TYPE=Release
cmake --build build-2c2g -j2
```

### 管理端

```bash
cd admin
npm install
npm run dev
```

### 移动端

```bash
cd frontend
flutter pub get
flutter run
```

## 4. 当前环境变量

## 8. 当前注意事项

- 发布版不允许默认连本地或占位域名。
- WebSocket 鉴权统一通过 query `token`。
- `server-lite` 是当前 2C2G 单机推荐模式。
- 改动接口或部署流程后必须同步更新文档。
