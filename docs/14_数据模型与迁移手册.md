## 4. 索引重点

当前重点索引集中在：

- `stones(status, created_at)`
- `stones(status, mood_type, created_at)`
- `paper_boats(stone_id / sender_id / receiver_id)`
- `temp_friends(user1_id / user2_id / expires_at)`
- `friend_messages(sender_id / receiver_id / pair_created)`
- `notifications(user_id, is_read, created_at)`
- `consultation_messages(session_id, created_at)`
- `reports(status, created_at)`
- `user_sessions(refresh_token_hash / user_id, refresh_expires_at)`

## 5. 当前模型目录

`backend/models/` 当前不放 ORM 类定义，而放运行期模型文件：

- `sentiment_zh.onnx`
- `vocab.txt`
- `sentiment_domain_lexicon.tsv`

这些文件主要服务：

- ONNX 情绪分析
- 词表加载
- 领域词典增强

## 6. 当前运维注意项

- 迁移以 `backend/start-docker.sh` / 容器启动时自动执行为主
- 当前 PostgreSQL 存在 `collation version mismatch`
- 当前高顺序扫描表仍需重点盯：
  - `stones`
  - `users`
  - `emotion_tracking`
  - `stone_embeddings`
